How to Use the Deep Desert Mapping Tool

Basic Controls

  • Left-click a subcell (3x3 grid within each cell) to open the icon picker.
  • Right-click a grid cell to toggle explored/unexplored status.
  • Double-click a grid cell to toggle explored/unexplored status.
  • Drag & Drop icons from the legend panels to place them on the map.
  • Escape key closes any open modal (icon picker or help).

Icon Management

  • Each cell is divided into a 3x3 subcell grid for precise icon placement.
  • Icons automatically scale based on quantity in the same subcell (1-4+ icons).
  • House icons can only be placed once per map and show as grayed out when already placed.
  • Click an icon in the picker again to remove it from the subcell.

Map Features

  • Explore All - Mark all cells as explored at once.
  • Undo/Redo - Revert or replay your last actions (up to 100 steps).
  • Clear Map - Reset the entire map (requires confirmation).
  • Unexplored cells appear with a dark overlay (70% opacity).
  • Grid coordinates are shown in each cell (A1-I9).

Export & Sharing

  • Export JSON - Save your map data to a file for backup.
  • Import JSON - Load a previously saved map file.
  • Export PNG - Generate a high-quality image with legends and timestamp.
  • Share Link - Create a shortened URL to share your map with others.
  • Combine Maps - Link to external tool for merging multiple maps.

Storm Timer

  • Shows countdown to the next storm for your selected server region.
  • Toggle between UTC and Local time display.
  • Storm phases: Normal, Active.
  • Supported regions: Europe, North America, South America, Asia, Oceania.

Tips & Shortcuts

  • The map auto-saves to your browser's local storage.
  • PNG exports include transparent legend boxes with all icons labeled.
  • The tool is mobile-responsive with adjusted layouts for smaller screens.
  • Icons in subcells support up to 4 in specific layouts, 5+ in dynamic grid.
